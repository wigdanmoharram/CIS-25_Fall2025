Week 6

This folder contains all C++ code files for Week 6 of CIS25.
Included Files:
- `hello_inventory.cpp` – prints "Hello Inventory System" and shows data types.
- `menu_if.cpp` – simple menu using if statements.
- `menu_functions.cpp` – menu calling functions.
- `bank_account.cpp` – BankAccount class with deposit, purchase, and transaction logging.
- `binary_search_100_items.cpp` – dynamic array of items with binary search.
- `inventory_class.cpp` – class to save and load items to/from file.
- `module10_inheritance.cpp` – demonstrates inheritance and method overriding.
- `module11_constructors_destructors.cpp` – demonstrates constructors and destructors.
- `module12Working with External.cpp` – reads and displays external CSV dataset.
